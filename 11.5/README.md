Switch 语句用于基于不同的条件来执行不同的动作。

  Switch(变量){
case 1:
如果变量和1的值相同，执行该处代码
break;
case 2:
如果变量和2的值相同，执行该处代码
break;
case 3:
如果变量和3的值相同，执行该处代码
break;
default:
如果变量和以上的值都不相同，执行该处代码
break;
}

switch后边的量和case后边值的数据类型必须保持一致。

自增自减

i++  ++i
在不参与运算的情况下，i++和++i都是在变量的基础加1
	var n1=123;
	++n1;
	alert(n1);

在参与运算的情况下
	var n1=123;
	var n2=++n1;
	alert(n2);
	alert(n1);

Var i=123;
Var j=i++;  先将i的值123赋值给j,之后再自增
j的值为123  i 的值为124
Var j=++i;  i先自增，在赋值给j
J的值为124，i的值 124
